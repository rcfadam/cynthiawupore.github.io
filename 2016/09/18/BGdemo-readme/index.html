BGdemo简略简介...

目录结构

  • css // 样式文件
  • fonts // 字体
  • img // 图片
  • js // js文件
  • jsonTest // 测试json
  • l10n // 国际化 语言
  • template //模板
  • tpl // html模板
  • index.html // 入口文件

主要目录

  • index.html //视图入口
  • app.js // js文件入口
  • controllers.js // controller文件

样式注意事项

  • 表单前文字不需要冒号
  • 必填项必须加星号
  • 条件尽量都按一行三个显示;少于三个的，一行显示，按钮放后面
  • 注意对齐！注意对齐！注意对齐！

布局

所有模块的html片段，都加在这里面：

<div class="wrapper-md">...</div>

表单

.form-horizontal 表单垂直排列

<div class="form-horizontal">...</div>

.form-group 模块 .has-error 报错样式

<form class="form-group has-error">...</form>

必填项的星号

<i class="asterisk">*</i>

网格 .col-sm-2 和 .col-sm-9 ; bootstrap讲网页显示区域分为12份，.col-sm-2指的是占两份。sm是小屏，md是中屏，lg是大屏。可参考boorstrap

<label class="col-sm-2 control-label"...</label>
<div class="col-sm-9">
     <input type="text" class="form-control">
     <span class="help-block m-b-none">提示文字</span>
</div>

.分割虚线

<div class="line line-dashed b-b line-lg"></div>

弹窗

data-target : 控制弹出窗的显示，写入弹出窗的id

data-keyboard : 布尔值，按键盘的ESC关闭弹窗

data-backdrop : 布尔值，时候包含半透明背景

注意：iframe里的提示框定位失效（在页面的最顶上）问题，目前解决方案：在弹出窗操作里添加 $(“html,body”).animate({“scrollTop”:0})

<button type="button" class="btn btn-info"  data-toggle="modal" data-target="#modalTest">点击出现弹窗</button>

</div>

统一提示框

在controller里面注入；ui.bootstrap.modal.transform

$scope.myAlert=function(){
    $rootScope.open({
      title: '信息提示', //提示框标题
      message: '',   //提示框内容
      isCancel: true,  //是否需要取消功能
      cancelText: '取消', //取消按钮文字

      OKText: '确定',       //确定按钮文字

      // scope:$scope
    })
